UndefinedTypes

Gets a collection of strings, each of which is an undefined type that applies to the DTDL element that corresponds to this object.

Declaration
public IReadOnlyCollection<string> UndefinedTypes { get; }
Property Value
Type Description
System.Collections.Generic.IReadOnlyCollection<System.String>

A collection of undefined type strings.
Remarks

An undefined type is any JSON-LD term used as a value of "@type&quot; that is not defined in the DTDL language and not defined by any DTDL extension included via the "@context&quot; property. The DTDL language permits undefined types to be co-typed on any DTDL element regardless of its primary type. The DTDL language permits undefined propertyes to be added to any DTDL element that has an undefined co-type.
